# 🛍️ Fashion E-Commerce UI (Static)

A modern, clean, and fully responsive multi-page e-commerce website built using only HTML and CSS.
This project focuses on UI/UX design, layout structure, and responsiveness without using JavaScript.

---

## 🚀 Features

- 📱 Fully Responsive Design (Mobile, Tablet, Desktop)
- 🎨 Clean and Modern UI
- 📃 Multi-Page Layout
- 🛒 E-commerce Components (Cart, Product, Filters, etc.)
- ⚡Lightweight (No frameworks, no JS)

---

## 📂 Pages Included
- 🏠 Home Page
- 📦 Product Detail Page
- 📁 Category Page
- 🛒 Cart Page

---

## 🛠️ Built With

- HTML 5
- CSS3 (Flexbox & Grid)

---

## 📱 Responsiveness

This project is fully responsive and optimized for:

- Mobile Devices 📱
- Tablets 📲
- Desktop Screens 💻

---

## 🎯 Purpose

This project was created to:

- Practice front-end layout design
- Improve CSS skills (Flexbox & Grid)
- Build a real-world e-commerce UI without JavaScript
